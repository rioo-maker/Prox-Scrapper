## Description

**Proxy Scraper** is a tool that automatically collects public proxy servers from multiple online sources and compiles them into a clean list.

Proxies allow users to route their internet traffic through another server, which can help with testing, anonymity, web scraping, or bypassing certain restrictions.

---

## Features

* Scrapes proxies from multiple public websites
* Supports common proxy types (HTTP / HTTPS / SOCKS)
* Removes duplicate proxies
* Fast and lightweight
* Saves proxies to a file for easy reuse

---

## Use Cases

This tool can be useful for:

* Web scraping projects
* Network testing
* Security research
* Learning about proxy infrastructure

---

## How it Works

1. The program requests proxy lists from multiple public sources.
2. It extracts proxy IP addresses and ports.
3. Duplicate proxies are removed.
4. The cleaned list is saved to a file.

Example format:

```
IP:PORT
192.168.1.10:8080
45.12.88.3:3128
103.44.12.7:1080
```
